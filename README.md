## How To Use

```bash
# Clone this repository
$ git clone https://github.com/everkers/vibeverse-ai

# Go into the repository
$ cd vibeverse-ai

# Install dependencies
$ npm install

# Add your env variables
$ cp env.example .env

# Run the app
$ npm start
```
